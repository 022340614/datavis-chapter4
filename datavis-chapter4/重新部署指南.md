# 重新部署操作指南

## 🚀 立即操作步骤

### 第一步：删除现有仓库
1. 登录GitHub
2. 访问：`https://github.com/022340614/datavis-chapter4/settings`
3. 滚动到页面最底部
4. 在"Danger Zone"找到"Delete this repository"
5. 输入 `datavis-chapter4` 确认删除

### 第二步：创建新仓库
1. 访问：`https://github.com/new`
2. 填写信息：
   - Repository name: `datavis-chapter4`
   - Description: `第4章数据可视化平台`
   - Public
   - 不初始化README（直接上传文件）
3. 点击"Create repository"

### 第三步：上传文件
1. 在新建的仓库页面
2. 点击"Add file" → "Upload files"
3. 将整个 `datavis-chapter4` 文件夹内容拖拽上传
4. 提交更改

### 第四步：启用GitHub Pages
1. 进入Settings → Pages
2. 配置：
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
3. 点击Save

### 第五步：等待部署
- 首次部署需要2-5分钟
- 查看Actions标签确认构建状态
- 访问：`https://022340614.github.io/datavis-chapter4/`
