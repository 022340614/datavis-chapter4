# 第4章 数据可视化平台

基于第4章"图表样式与颜色应用"内容创建的公共数据可视化平台，展示了多种数据可视化技术和交互式图表。

## 🚀 快速开始

### 本地运行
1. 双击 `index.html` 文件在浏览器中打开
2. 或使用本地服务器：`python -m http.server 8000`

### GitHub Pages部署
1. 将所有文件上传到GitHub仓库 `datavis-chapter4`
2. 在仓库Settings中启用GitHub Pages
3. 访问：`https://022340614.github.io/datavis-chapter4/`

## 📊 平台功能

### 1. 图书采购情况分析
- **技术应用**：颜色应用、堆叠柱状图
- **数据来源**：两个地区对不同种类图书的采购情况
- **特色**：使用对比色展示地区差异

### 2. 汇率走势分析
- **技术应用**：不同线型、颜色对比
- **数据来源**：2017年与2019年美元/人民币汇率对比
- **特色**：实线与虚线对比，清晰展示趋势变化

### 3. 产品销售趋势
- **技术应用**：数据标记、多种标记样式
- **数据来源**：不同产品各季度销售额
- **特色**：菱形、三角形、矩形多种标记样式

### 4. 温度走势预测
- **技术应用**：字体设置、数据标记
- **数据来源**：未来15天最高最低气温预测
- **特色**：温度数值标注，清晰展示温差

### 5. 科赫雪花分形
- **技术应用**：填充区域技术
- **技术实现**：递归算法生成分形图案
- **特色**：数学可视化，展示分形几何美

### 6. 颜色映射演示
- **技术应用**：颜色搭配和应用
- **数据来源**：不同色系的应用效果
- **特色**：现代化配色方案展示

## 🛠 技术栈

- **前端框架**：纯HTML5 + CSS3 + JavaScript
- **图表库**：Chart.js 3.x
- **响应式设计**：CSS Grid + Flexbox
- **部署平台**：GitHub Pages
