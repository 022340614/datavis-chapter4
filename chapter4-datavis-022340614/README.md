# 第4章 数据可视化平台

基于第4章"图表样式与颜色应用"内容创建的公共数据可视化平台，展示了多种图表样式和颜色应用技巧。

## 平台功能

### 1. 图书采购情况可视化
- 使用堆叠柱状图展示两个地区对不同种类图书的采购情况
- 应用了不同的颜色方案（#FFCC00和#B0C4DE）
- 包含网格线和图例

### 2. 汇率走势分析
- 折线图对比2017年7月与2019年7月美元/人民币汇率
- 使用不同线型（实线和虚线）区分年份
- 湖绿色（#006374）和紫色（#8a2e76）配色方案

### 3. 产品销售趋势
- 展示三种产品在不同季度的销售额
- 使用不同标记样式（菱形、三角形、矩形）
- 多种线型组合应用

### 4. 温度走势预测
- 显示未来15天最高和最低气温变化
- 包含数据标记和温度标注
- 响应式设计适配不同屏幕尺寸

### 5. 科赫雪花分形
- 使用Canvas技术绘制彩色雪花图案
- 应用填充区域技术（lightsalmon和orangered）
- 展示数学分形的美感

### 6. 颜色映射演示
- 展示本章使用的各种颜色
- 包含十六进制代码显示
- 颜色对比和搭配示例

## 技术特点

### 前端技术栈
- **HTML5**: 语义化结构
- **CSS3**: 现代样式设计，包含渐变、动画、响应式布局
- **JavaScript**: 交互功能实现
- **Chart.js**: 专业图表库
- **Canvas**: 自定义图形绘制

### 设计原则
- **响应式设计**: 适配桌面和移动设备
- **用户体验**: 平滑滚动、悬停效果、导航高亮
- **可访问性**: 清晰的字体、合适的对比度
- **性能优化**: 按需加载、动画优化

## 运行方式

### 方式一：直接打开
1. 双击 `index.html` 文件
2. 在浏览器中查看可视化效果

### 方式二：本地服务器
```bash

python -m http.server 8000

# 或使用Node.js
npx http-server
```

### 方式三：在线部署
可将文件上传至GitHub Pages、Netlify等静态网站托管服务
